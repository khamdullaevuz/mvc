# mvc
### simple mvc project
# Usage
## Using git (for usage and contribute)
### Clone repository
```shell
git clone https://github.com/khamdullaevuz/mvc
```
### Dump autoloader
```shell
composer dump-autoload
```
## Using composer (for usage)
### Create project
```shell
composer create-project khamdullaevuz/mvc:^dev-main
```
